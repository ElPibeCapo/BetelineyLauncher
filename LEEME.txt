COMO COMPILAR (LINUX — Nobara / Fedora / Ubuntu / Arch):
1. cd BetelineyLauncher/source
2. bash COMPILAR_LINUX.sh   → [4] instalar deps (solo primera vez)
3. bash COMPILAR_LINUX.sh   → [1] build completo (~10-30 min primera vez)
4. bash COMPILAR_LINUX.sh   → [2] recompilar rapido (usa esto siempre)
5. El ejecutable queda en:  build/beteliney

COMO COMPILAR (WINDOWS):
1. Doble clic en COMPILAR_BETELINEY.bat
2. Primera vez: elige [1] build completo
3. Siempre despues: elige [2] recompilar rapido
4. El .exe queda en: build\beteliney.exe
   (la carpeta se abre sola al terminar)

LANZAR RAPIDO (Linux):
  bash lanzar.sh
  (compila automaticamente si falta el ejecutable)

ACTUALIZAR (Linux):
  bash beteliney-updater.sh
  Flags: --check / --silent / --force / --appimage
         --log [N]   → muestra los últimos N logs de update (default 1)

DISTRIBUIR:
  Windows: EMPAQUETAR_WINDOWS.bat → [1] ZIP portable
  Linux:   EMPAQUETAR_LINUX.sh  → [1] tar.gz portable (sin FUSE)
                                 → [2] AppImage
                                 → [3] .deb (Debian/Ubuntu)
           EMPAQUETAR_APPIMAGE.sh → AppImage standalone (auto-descarga
                                     linuxdeploy/appimagetool si faltan)

MODOS DE COMPILAR_LINUX.sh:
  [1] Build completo    — configura + compila (primera vez)
  [2] Recompilar rapido — incremental (usa esto siempre)
  [3] Limpiar + rebuild — borra todo y recompila (si algo esta raro)
  [4] Solo instalar deps
  [5] Crear AppImage    — distribuible sin instalar nada

MODOS DE COMPILAR_BETELINEY.bat:
  [1] Build completo    — configura + compila (primera vez)
  [2] Recompilar        — solo compila lo que cambio (rapido)
  [3] Limpiar + rebuild — borra todo y recompila
  [4] Solo configurar   — cmake configure sin compilar
  [5] Debug build       — con simbolos de depuracion (sin LTO)

REQUISITOS WINDOWS (instalar antes de compilar):
  - Qt 6.x con MinGW 64-bit
    https://www.qt.io/download-qt-installer
    Marcar: Qt 6.x → MinGW 64-bit
            Qt Tools → Ninja + CMake
  - Java 21 (Eclipse Temurin recomendado)
    https://adoptium.net
  - vcpkg incluido en vcpkg_local/ (no requiere instalacion extra)
  - Git (para el updater)

ADVERTENCIA — MinGW vs MSVC:
  Este proyecto compila con MinGW (GCC). Son ABI incompatibles con MSVC.
  NO mezcles DLLs de un build MSVC con este build MinGW → crashes.
  El bat copia automaticamente las DLLs correctas de MinGW.

PROBLEMAS FRECUENTES:

  StackOverflowError con mods
    → Agregar -Xss2M en argumentos JVM de la instancia
    → O usar perfil "Balanceado" que ya incluye -Xss1M

  Lag / stutters cada 5-15 segundos
    → Usar el perfil JVM adecuado (ver docs/PERFILES_JVM.md)
    → No asignar más del 60% de RAM total
    → En Vega 10: usar perfil iGPU (máx 1.5 GB)

  Crash al iniciar en Vega 10
    → bash lanzar.sh --debug para ver el error exacto
    → Verificar que Qt6 está instalado correctamente

  CMake Error: Qt6 not found
    → bash COMPILAR_LINUX.sh → [4] instalar deps

  tomlplusplus not found
    → El script ya pasa -Dtomlplusplus_DIR automáticamente
    → Si compilás manualmente: agregar ese flag al cmake

  NTFS partición en read-only
    → sudo mount -t ntfs-3g /dev/sdXY /mnt/windows
    → O desactivar Fast Startup/Hibernate en Windows:
      powercfg /h off   (ejecutar como Admin en Windows)

  GPU integrada no detectada para perfil JVM
    → El launcher usa lspci para detección automática
    → Si falla: seleccionar "iGPU / RAM compartida" manualmente
    → Guía completa: docs/TROUBLESHOOTING.md
